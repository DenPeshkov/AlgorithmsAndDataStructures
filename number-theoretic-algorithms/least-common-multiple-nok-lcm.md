# Least common multiple \(НОК/LCM\)

```java
int lcm(int a, int b) {    
  return a / gcd(a, b) * b;
}
```

| Space | Time |
| :--- | :--- |
| $$O(1)$$  | $$O(log\ min(a,b))$$  |

{% hint style="info" %}
 Здесь выгодно сначала поделить на $$gcd$$, а только потом домножать на $$b$$, поскольку это поможет избежать переполнений в некоторых случаях
{% endhint %}

{% hint style="info" %}
 Вычисление наименьшего общего кратного нескольких чисел может быть также сведено к нескольким последовательным вычислениям НОК от двух чисел: $$lcm(a,b,c)=lcm(lcm(a,b),c)$$ 
{% endhint %}
