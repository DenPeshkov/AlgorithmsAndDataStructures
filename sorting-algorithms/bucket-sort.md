# Bucket sort

```java
// k is the number of buckets
static void bucketSort(int[] arr, int k) {
  List<Integer>[] buckets = new List[k];
  
  for (int i = 0; i < k; i++) buckets[i] = new ArrayList<>();
  
  int max = findMax(arr);
  
  for (int el : arr) buckets[hash(el, max, k)].add(el);
  
  for (List<Integer> bucket : buckets) {
    bucket.sort(Comparator.naturalOrder());
  }
  
  int i = 0;
  for (List<Integer> bucket : buckets) for (int el : bucket) arr[i++] = el;
}

private static int findMax(int[] arr) {
  int max = arr[0];
  
  for (int i = 1; i < arr.length; i++) {
    if (arr[i] > max) max = arr[i];
  }
  
  return max;
}

private static int hash(int el, int max, int numberOfBuckets) {
  return (int) ((double) el / max * (numberOfBuckets - 1));
}
```

| Space | Time |
| :--- | :--- |
| $$O(n+k)$$  | $$\Omega(n^2)\ \Theta(n+k)\ \ O(n+k)$$  |
